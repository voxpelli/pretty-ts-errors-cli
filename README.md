## Install

### Globally

```sh
npm install -g @voxpelli/pretty-ts-errors-cli
```

### Locally

```sh
npm install -D @voxpelli/pretty-ts-errors-cli
```

## Usage

Terminal output:

```sh
cat examples/input.txt | pretty-ts-errors
```

Markdown output:

```sh
cat examples/input.txt | pretty-ts-errors -m
```
